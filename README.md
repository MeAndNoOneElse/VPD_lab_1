##  Архитектура

```
┌─────────────────────────────────────────────────────────────┐
│                    ЭКСПЕРИМЕНТ                              │
├─────────────────────────────────────────────────────────────┤
│  LEGO EV3 (ev3dev2)                                         │
│  Python 3: управление мотором, логирование U, θ, ω          │
│  ↓                                                          │
│  Массив файлов: data[U].txt (10 напряжений ±20..±100%)      │
└─────────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────────┐
│               ОБРАБОТКА ДАННЫХ (MATLAB)                     │
├─────────────────────────────────────────────────────────────┤
│  Чтение временных рядов                                     │
│  ↓                                                          │
│  Нелинейная аппроксимация:                                  │
│    θ(t) = K·U·(t - Tm·(1 - exp(-t/Tm)))                     │
│    ω(t) = K·U·(1 - exp(-t/Tm))                              │
│  ↓                                                          │
│  Определение параметров: K, Tm                              │
│  ↓                                                          │
│  Графики: углы, скорости, аппроксимация, сравнение          │
└─────────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────────┐
│            Симкляция МОДЕЛИ (Simulink)                      │
├─────────────────────────────────────────────────────────────┤
│  Блоки: Sum, Gain, Integrator (K/(1+Tm·s))                  │
│  Исходные данные: U (напряжение)                            │
│  Выход: θ, ω → сравнение с экспериментом                    │
└─────────────────────────────────────────────────────────────┘
```

---

### 2️⃣ На ноуте (MATLAB обработка)

```matlab
m_code_lab_1

% 1. Читает 10 data[U].txt файлов
% 2. Аппроксимирует кривые
% 3. Строит графики в graphs/
% 4. Выводит параметры K, Tm, Ke, Km
% 5. Прогоняем модель в Simulink и сравниваем
```

**Результаты в папке `graphs/`:**
```
angles/              — кривые углов для каждого U
velocities/          — кривые скоростей для каждого U
comparison/          — эксперимент vs аппроксимация
parameters/          — итоговые графики K, Tm, ke
all_results.csv      — таблица всех параметров
```
